# AI-Driven Primary Care Experience
This repository contains my  academic project for design thinking course (2023-2024)

## Content 
- Presentation PDF
- Details of the project and methodology
- Implemented version of idea

## Details of the Project 
- Project Name: HealthHub
- In collaboration with Mediclinic Group
- Main activites: 
  - Mapped current-state customer pain points and touchpoints across primary care,
    applying Change of Meaning methodology to redesign the end-to-end patient
    experience around accessibility, personalization, and AI-augmented care
  - Designed an multichannel healthcare solution combining a physical AI-guided capsule
    with a connected mobile app, developing human-centered AI design principles
    balancing automation with human oversight
  - Conducted stakeholder mapping across direct players, professionals, and cultural
    interpreters to validate solution feasibility and identify ecosystem dynamics
